## Install
```bash
$ pip install sphinx
$ pip install sphinx_rtd_theme
```

## Develop the docs
1. Go to the root directory and run:
  ```bash
  $ make documentation
  ```
  OR
  ```bash
  $ make
  ```
2. Then run
  ```bash
  $ make open-docs
  ```
