causal_data_augmentation.experiment_api¶

class CausalDataAugmentationEagerTrainingExperimentAPI(augmenter_config, fit_to_aug_only, aug_coeff=None, debug=False)¶

Bases: object

The interface class where the method’s codes are integrated for the experiment.

Constructor.

apply_augmenter(augmenter_config, method, data, admg)¶

Perform the augmentation using the augmenter configured by augmenter_config.

Parameters

  • augmenter_config (AugmenterConfig) – Method configuration.

  • method (EagerCausalDataAugmentation) – Instantiated method object.

  • data (DataFrame) – Data to be augmented.

  • admg (ADMG) – ADMG to be used for the augmentation.

Return type

Tuple[DataFrame, ndarray]

Returns

Tuple containing

  • augmented_data : The augmented data DataFrame.

  • aug_weights : The instance weights corresponding to the augmented data.

run_method_and_eval(data, graph, predicted_var_name, predictor_model, evaluators, run_logger)¶

Run the method and record the results.

Parameters

  • data (DataFrame) – The data to be augmented.

  • graph (Tuple[List[str], List[Tuple[str, str]], List[Tuple[str, str]]]) – The ADMG object used for performing the augmentation.

  • predicted_var_name (str) – The name of the predicted variable.

  • predictor_model – Trainable predictor model to be trained on the augmented data. Should implement fit() and predict().

  • evaluators (Iterable[Evaluator]) – a series of evaluators applied to the trained predictor.

  • run_logger – The logger to record the experiment.
