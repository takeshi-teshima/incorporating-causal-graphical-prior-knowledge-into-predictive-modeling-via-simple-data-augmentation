class AugmenterKernel(v_names, c_names, c_weighter)¶

Bases: object

Class for holding the weight computer. Corresponds to $p(v | c)$.

Parameters

  • v_names (List[str]) – names of the variables to be augmented ($v$).

  • c_names (List[str]) – names of the variables to be conditioned ($c$).

  • c_weighter (WeightComputer) – kernel object used for the conditional resampling. This object needs to have been already fit to the parameters.

summarize_duplicates_df(df, key, method='sum')¶

Perform contraction of duplicating entries while aggregating one of the columns.

Parameters

  • df (DataFrame) – DataFrame to summarize.

  • key (str) – The key to be aggregated.

  • method (str) – Method to use for aggregating the key column.

Return type

DataFrame

Returns

New DataFrame.

Example

>>> A = pd.DataFrame.from_dict([{'A': 1}, {'A': 1}, {'A': 2}, {'A': 2}])
>>> B = pd.DataFrame.from_dict([{'B': 1}, {'B': 3}, {'B': 5}, {'B': 7}])
>>> df = pd.concat((A, B), axis=1)
>>> df
   A  B
0  1  1
1  1  3
2  2  5
3  2  7
>>> summarize_duplicates_df(df, 'B', 'sum')
   A   B
0  1   4
2  2  12
