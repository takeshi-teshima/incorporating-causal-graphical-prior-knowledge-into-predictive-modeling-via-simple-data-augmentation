class ADMGTianFullAugmenter(graph, top_order=None)¶

Bases: causal_data_augmentation.causal_data_augmentation.augmenter.admg_tian_augmenter.base.ADMGTianAugmenterBase

Proposed method that augments the training data to the maximum possible extent.

Constructor.

Parameters

  • graph (ADMG) – the ADMG model to be used for the data augmentation.

  • top_order (Optional[List[str]]) – a valid topological order on the graph (a list of the vertex names). If None is provided, it is automatically computed from the graph (default: None).

augment(weight_threshold, weight_threshold_type, normalize_threshold_by_data_size)¶

Perform Tian factorization and augment the data accordingly.

Parameters

  • weight_threshold (Optional[float]) – the lower weight threshold for performing the augmentation.

  • normalize_threshold_by_data_size (bool) – whether to normalize the threshold by the data size. (True: divide the threshold by the data size)

Return type

Tuple[DataFrame, ndarray]

Returns

Tuple containing

  • augmented_data : the fully augmented data.

  • weights : the instance weights based on the kernel values.
