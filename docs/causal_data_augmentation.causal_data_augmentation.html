causal_data_augmentation.causal_data_augmentation¶

causal_data_augmentation.causal_data_augmentation.api¶

Versions of the method (corresponding to different modeling choices, subroutines, etc.) will be defined in this layer.

class EagerCausalDataAugmentation(method_config=FullAugment(weight_threshold=None, weight_threshold_type='total', normalize_threshold_by_data_size=True, weight_kernel_cfg={}))¶

Bases: object

Implementation of Causal Data Augmentation. Augments the data and returns the augmented data (i.e., not lazy = eager). Suitable for those predictor classes that go better with one-time data augmentation than on-the-fly augmentation.

Constructor.

Parameters

method_config (AugmenterConfig) – the config of the method.

augment(data, estimated_graph)¶

Generate augmented data. Does not consider overlapping, etc., against the original data.

Parameters

  • data (DataFrame) – The source domain data to be used for fitting the novelty detector.

  • estimated_graph (ADMG) – The ADMG object used for performing the augmentation.

Returns

  • Tuple of (augmented_data, weights) : if self.sampling_method is 'full'.

  • augmented_data : if self.sampling_method is 'stochastic'.

Return type

One of the following

Examples

>> weight_threshold = 1e-5 >> augmenter = EagerCausalDataAugmentation(FullAugment(weight_threshold)) >> raise NotImplementedError()
